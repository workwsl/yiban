译伴 · 双语网页翻译

使用 DeepSeek、Qwen 或 OpenAI-compatible 大模型，在网页原文下方插入译文，不破坏原有布局。支持划词翻译、输入翻译与自定义 AI 专家。

下载 v0.2.0 查看源码

核心功能

轻量接入，本地配置，按需翻译。

  • 整页双语翻译

    扫描页面文本节点，在原内容下方插入译文，保留原有排版结构。

  • 划词翻译

    选中文字后出现浮动按钮，快速查看单段译文。

  • 输入翻译

    独立翻译页，实时输入自动翻译，译文支持 Markdown 渲染。

  • 快捷键切换

    按 Alt + A 开启或停止当前页翻译。

  • 多模型支持

    DeepSeek、Qwen、任意 OpenAI-compatible 接口，可自定义 Base URL 与模型。

  • AI 专家提示词

    内置通用与技术文档风格，支持自定义提示词与覆盖配置。

  • 翻译缓存

    本地 IndexedDB 缓存，减少重复 API 调用，提升二次访问速度。

  • 网站黑名单

    指定域名永不自动翻译，灵活控制翻译范围。

安装与配置

从 GitHub Releases 下载构建包，加载到浏览器即可使用。

安装步骤

  1. 下载扩展包

    从 Releases 下载 yiban-v0.2.0.zip 并解压。

  2. 加载扩展

    打开 chrome://extensions/ 或 edge://extensions/，开启「开发者模式」，点击「加载已解压的扩展程序」，选择解压后的目录。

  3. 配置模型

    点击扩展图标 →「设置」，在「模型配置」中填写 API Key（DeepSeek / Qwen 预设或自定义 OpenAI-compatible），测试连接后设为默认。

  4. 开始翻译

    在 Popup 中选择目标语言与 AI 专家，点击「翻译当前页」；或使用划词翻译、输入翻译页。

隐私与安全

API Key 与配置仅保存在你的浏览器本地。

  • API Key 与翻译配置保存在 chrome.storage.local，不上传到第三方服务器。
  • 翻译请求由扩展 Background Service Worker 发起，网页脚本无法读取 Key。
  • 翻译缓存存储在本地 IndexedDB，不会上传到第三方（模型 API 请求除外）。
